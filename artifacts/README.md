# Konosuba WhatsApp Bot Platform

A premium anime-inspired WhatsApp bot management platform with dark fantasy aesthetics.

## Structure

```
/server       — Express API (Node.js + MongoDB)
/website      — Landing page + user dashboard (React + Vite)
/bot-manager  — Admin control panel (React + Vite)
```

## Deploy on Render (One-Click)

1. Push this repo to GitHub
2. Go to [render.com](https://render.com) → New → Blueprint
3. Connect your GitHub repo — Render reads `render.yaml` automatically
4. Set the environment variables:
   - `MONGO_URI` — Your MongoDB Atlas connection string
   - `ADMIN_PASSWORD` — Admin panel password
   - `BOT_WEBHOOK_SECRET` — (optional) shared with your bot
   - `JWT_SECRET` is auto-generated by Render

## Manual Deploy

### Environment Variables (Render Dashboard)
| Variable | Description |
|----------|-------------|
| `MONGO_URI` | MongoDB Atlas URI |
| `JWT_SECRET` | Long random secret for JWT signing |
| `ADMIN_PASSWORD` | Password for admin panel |
| `BOT_WEBHOOK_SECRET` | Optional webhook secret |

### Build Command
```
cd website && npm install && npm run build && cd ../bot-manager && npm install && npm run build && cd ../server && npm install
```

### Start Command
```
node server/server.js
```

## Routes
- `/` — Landing page
- `/auth` — Login / Register  
- `/dashboard` — User dashboard (requires login)
- `/manager` — Admin bot manager (requires admin key)
- `/api/*` — REST API

## Bot Integration
Point your Baileys bot to POST user data to `/api/website/*` with `x-admin-key` header set to your `BOT_WEBHOOK_SECRET`.
